How to Add the Bot to Your Discord Server

  1. Make sure you are logged into your Discord account.
  2. Go to the bot's authorization page using the link below.
  3. Select the Discord server where you want to add the bot.
  4. Click the "Authorize" button to add the bot to the selected server.
  5. Complete the CAPTCHA to confirm you are not a robot.
  6. Once the bot is added, it will be available on the server and ready to use.
